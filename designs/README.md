# Designs

Εδώ αποθηκεύονται τα αρχεία που αφορούν τα γραφικά και τον οπτικό σχεδιασμό του site, όπως mockups, screenshots, εικονίδια και visual references.

- `log-lines-context-designs.html` — Δ' γύρος Καταγραφής: η επιλεγμένη Ιδέα 1 «Οι Γραμμές» με τη σημερινή γραμματοσειρά στα πεδία (χωρίς μπλε χειρόγραφη) και με ορατές τις υπόλοιπες κάρτες ασκήσεων· δύο εκδοχές — συρτάρι (οι διπλανές ξεμυτίζουν στα άκρα) και στοίβα (οι επόμενες προεξέχουν από πίσω).
- `log-card-interior-designs.html` — Γ' γύρος Καταγραφής: το εσωτερικό της κάρτας άσκησης. Πέντε διαδραστικές ιδέες (γραμμές τετραδίου με κουμπάκι μονάδας, σφραγίδα σετ, κοντέρ −/＋, καρμπόν προηγούμενης προπόνησης, ενσωματωμένο αριθμητήριο) + τοίχος με 8 σκίτσα· διάγνωση: η κάρτα είναι φόρμα ενώ πρέπει να είναι σελίδα τετραδίου.
- `log-swipe-drawer-designs.html` — Β' γύρος της Καρτελοθήκης (επιλεγμένη Πρόταση 3): τέσσερις παραλλαγές περιβλήματος (tabs αρχείου, καβαλάρηδες, καθαρό συρτάρι, κάρτα-φάντασμα «＋») με την κάρτα άσκησης πιστό αντίγραφο του σημερινού styles.css — η κάρτα δεν αλλάζει καθόλου.
- `log-swipe-exercise-designs.html` — τέσσερις διαδραστικές προτάσεις για swipe ασκήσεων στην Καταγραφή (τράπουλα καρτελών, σπιράλ page-flip, καρτελοθήκη με tabs, χάρακας-ράγα)· καλύπτουν και τα δύο σενάρια (αντιγραφή + «Προσθήκη άσκησης» με effect προς τα δεξιά).
- `personal-records-progress-stamp-designs.html` — τρεις διαδραστικές προτάσεις για τη μεταφορά των καλύτερων επιδόσεων στην Επίβλεψη, κάτω από το γράφημα.
- `progress-reps-course-designs.html` — πέντε προτάσεις για το γράφημα της Επίβλεψης ώστε να φαίνεται και η πορεία των επαναλήψεων (ευθυγραμμισμένα πάνελ, αγκύλες κύκλου, εναλλαγή μετρικής, δείκτης 100, ζώνη κελιών)· χρώματα σειρών validated (#8a6a1c κιλά / #1f6493 επαναλήψεις).
