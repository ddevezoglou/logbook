# Logbook

Το **Logbook** είναι ένα απλό ημερολόγιο προπόνησης για να οργανώνεις τα προγράμματά σου, να καταγράφεις κάθε προπόνηση και να παρακολουθείς την πρόοδό σου.

## Χρησιμοποίησε την εφαρμογή

### [Άνοιξε το Logbook](https://ddevezoglou.github.io/logbook/)

Μπορείς να συνδεθείς με email ή Google. Τα δεδομένα αποθηκεύονται πρώτα στη συσκευή σου και συγχρονίζονται με τον λογαριασμό σου όταν υπάρχει σύνδεση.

## Βασικές δυνατότητες

- Πολλαπλά προγράμματα και κύκλοι προπόνησης 3–10 ημερών.
- Καταγραφή ασκήσεων, σετ, επαναλήψεων, βαρών και σημειώσεων.
- Ιστορικό, προσωπικά ρεκόρ και γραφήματα προόδου.
- Χρήση και χωρίς σύνδεση μετά την πρώτη φόρτωση.
- Ελληνικό, αγγλικό, γαλλικό και γερμανικό περιβάλλον.
- Συγχρονισμός ανάμεσα στις συσκευές σου, με revision checks και αυτόματη ασφαλή συγχώνευση.
- Ιδιωτικά ημερήσια cloud recovery snapshots με διατήρηση 30 ημερών.

## Εγκατάσταση στο κινητό

Άνοιξε το Logbook από τον browser του κινητού σου:

- **Android / Chrome:** μενού `⋮` → **Εγκατάσταση εφαρμογής**.
- **iPhone / Safari:** **Κοινοποίηση** → **Προσθήκη στην οθόνη Αφετηρίας**.

Για σύνδεση με Google, χρησιμοποίησε κανονικό Chrome, Edge ή Safari και όχι τον ενσωματωμένο browser άλλης εφαρμογής.
