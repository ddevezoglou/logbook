# Logbook — Ημερολόγιο Προπόνησης

Πρώτο λειτουργικό draft μιας εφαρμογής για καταγραφή προπονήσεων και δημιουργία εβδομαδιαίου προγράμματος.

## Εκτέλεση στον υπολογιστή

Δεν χρειάζεται εγκατάσταση. Άνοιξε το `index.html` με διπλό κλικ ή, για καλύτερη τοπική λειτουργία, τρέξε μέσα από τον φάκελο:

```powershell
npx.cmd serve .
```

και άνοιξε τη διεύθυνση που θα εμφανιστεί.

## Τι περιλαμβάνει

- Ονομασμένη προπόνηση ανά ημέρα με δυναμικό αριθμό ασκήσεων
- Εργάσιμα σετ και τεχνικά cues ανά προγραμματισμένη άσκηση
- Πολλαπλές ανεξάρτητες εβδομαδιαίες ρουτίνες, καθεμία με δικό της board ημερών και ασκήσεων
- Οι ημέρες δεσμεύονται μόνο μέσα στη ρουτίνα τους και ανοίγουν από την κάρτα τους για επεξεργασία
- Ορισμός ενεργού προγράμματος με αστέρι και αυτόματη εμφάνισή του στην Καταγραφή
- Εναλλακτική ελεύθερη προπόνηση χωρίς προϋπάρχον πρόγραμμα
- Επιλογή αριθμού σετ ανά ελεύθερη άσκηση και διαγράψιμα extra σετ στις προγραμματισμένες ασκήσεις
- Επίβλεψη με κάρτες ιστορικού, αριθμό εκτελεσμένων ασκήσεων, ρυθμό 7 ημερών και προσωπικά καλύτερα
- Τοπική αποθήκευση στον browser και αυτόματη μεταφορά του παλιού ενιαίου προγράμματος σε εβδομαδιαία ρουτίνα
- Διαγραφή εγγραφών και πλήρης προσαρμογή σε κινητό/υπολογιστή

## Σημαντικό για το πρώτο draft

Τα δεδομένα αποθηκεύονται μόνο στον συγκεκριμένο browser (`localStorage`). Δεν υπάρχει ακόμη λογαριασμός χρήστη, cloud συγχρονισμός ή βάση δεδομένων.

## Επόμενα λογικά βήματα

1. Επιλογή άσκησης από προσωπική βιβλιοθήκη αντί για ελεύθερο κείμενο.
2. Ομαδοποίηση πολλών ασκήσεων σε μία προπόνηση.
3. Επεξεργασία παλιών εγγραφών και αντιγραφή προηγούμενης προπόνησης.
4. Πρόοδος ανά άσκηση με γραφήματα κιλών, επαναλήψεων και όγκου.
5. Λογαριασμός χρήστη και online βάση δεδομένων.
6. Μετατροπή σε installable PWA και, αργότερα, mobile app.
